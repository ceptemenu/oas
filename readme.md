# Cepte Menu OAS

[API Documentation](https://ceptemenu.github.io/oas/) deployed to GitHub pages.

---

Install dependencies.

```shell
bun install
```

Build open api file.

```shell
bun run build
```

Lint source.

```shell
bun run test
```

Preview with built-in server.

```shell
bun run preview
```

Generate HTML file.

```shell
bun run html
```

Clean current dist folder.

```shell
bun run clean
```
